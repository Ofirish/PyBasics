דוגמאות ללולאות - עבור לדף pyloops

יסודות: משתנים והדפסה

אחסון מידע - משתנים (Variables)

משתנה הוא כלי לאחסון מידע בתוך התוכנית. ניתן לחשוב עליו כעל תווית שמצביעה על ערך מסוים. הפונקציה print() משמשת להצגת המידע על המסך.

📝 דוגמה:
שם_משתמש = "דוד"
גיל = 32

# f-string מאפשר שילוב של טקסט ומשתנים
print(f"השם הוא {שם_משתמש} והגיל הוא {גיל}.")
💡 טיפ:

מומלץ לתת למשתנים שמות משמעותיים שמסבירים מה הם מכילים. זה עוזר לקרוא ולהבין את הקוד אחר כך.

קלט משתמש ופעולות מתמטיות

קבלת נתונים מהמשתמש (input())

הפונקציה input() מאפשרת לקבל מידע מהמשתמש בזמן ריצת התוכנית. חשוב לזכור: כל קלט שמתקבל הוא מסוג טקסט (string). אם רוצים להשתמש בו כמספר, צריך להמיר אותו בעזרת int() (למספר שלם) או float() (למספר עשרוני).

📝 דוגמה:
# קבלת קלט מהמשתמש ושמירה במשתנה
שם = input("מה שמך? ")
print(f"שלום, {שם}!")

# המרת קלט למספר שלם
מספר_כסף = int(input("כמה כסף יש לך? "))
print(f"עוד 5 שקלים זה: {מספר_כסף + 5}")

🧮 פעולות מתמטיות

פייתון תומכת בכל הפעולות המתמטיות הבסיסיות. הנה הנפוצות ביותר:

📝 דוגמה:
מספר1 = 10 
מספר2 = 3

# חיבור: +
תוצאת_חיבור = מספר1 + מספר2  

# חזקה: ** (כפל חוזר)
תוצאת_חזקה = מספר1 ** 2 

# מודולו: % (שארית חלוקה)
# 10 לחלק ל-3 זה 3 עם שארית 1
תוצאת_מודולו = מספר1 % מספר2 

print(f"תוצאת המודולו: {תוצאת_מודולו}")

בקרת זרימה: משפטי תנאי ולולאות

משפטי תנאי (if, elif, else)

משפטי תנאי מאפשרים לתוכנית לקבל החלטות לפי תנאים שונים. המבנה הבסיסי כולל: if (אם התנאי מתקיים), elif (אחרת, אם תנאי אחר מתקיים), else (בכל מקרה אחר).

📝 דוגמה:
צבע_רמזור = "צהוב" 

if צבע_רמזור == "אדום":
    print("עצור!")
elif צבע_רמזור == "צהוב":
    print("התכונן...")
else:
    print("סע!")

שילוב: לולאות תנאי (for + if/elif/else)

לולאת for מאפשרת לחזור על פעולה מספר פעמים, ובכל איטרציה ניתן לבדוק תנאים שונים.

📝 דוגמה:
מחזור_אורות = ["אדום", "צהוב", "ירוק"]

for צבע in מחזור_אורות:
    if צבע == "אדום":
        print(f"🔴 {צבע}: עוצרים.")
    elif צבע == "צהוב":
        print(f"🟡 {צבע}: מתכוננים.")
    else: 
        print(f"🟢 {צבע}: נוסעים.")

כלים שימושיים: הערות ואינדנטציה

הערות בקוד (# - Comments)

הערות בקוד מתחילות בסימן #. המחשב מתעלם מהן לחלוטין - הן נועדו עבורנו, המתכנתים, כדי לתעד ולהסביר את מה שהקוד עושה.

📝 דוגמה:
# המשתנה הזה מייצג את גובה המשתמש
גובה = 175 

if גובה > 140:
    # בודק אם מותר לעלות למתקן
    print("מותר לעלות")
💡 טיפ:

הערות טובות הן קריטיות לתחזוקת קוד. הן עוזרות לנו ולאחרים להבין את הלוגיקה גם אחרי זמן רב.

אינדנטציה (Indentation) - הזחה

אינדנטציה היא אחת התכונות הייחודיות והחשובות ביותר בפייתון. בפייתון, ריווחים ותחילת שורה קובעים את המבנה של הקוד, בניגוד לשפות אחרות שמשתמשות בסוגריים מסולסלים.

📝 דוגמה נכונה:
│if True:
│└──print("זה עובד!") # ← 4 רווחים או Tab
│└──print("גם זה") # ← אותה רמת הזחה
⚠️ שגיאה נפוצה:
│if True:
│print("זה יגרום לשגיאה!") # ← אין הזחה

פייתון תזרוק שגיאת IndentationError במקרה הזה!

💡 כללי הזהב לאינדנטציה:
  • השתמש ב-4 רווחים לכל רמת הזחה (זו המוסכמה המקובלת)
  • אל תערבב רווחים ו-Tab - בחר אחד ושמור עליו
  • קוד באותה רמת הזחה שייך לאותו בלוק
  • אינדנטציה קובעת מה שייך ל-if, for, while, פונקציות, וכיתות
📝 דוגמה מורכבת יותר:
גיל = 15

if גיל >= 18:
    print("אתה בוגר")
    print("מותר לך להצביע")
else:
    print("אתה עדיין קטין")
    if גיל >= 16:
        print("אבל מותר לך לעבוד")
    else:
        print("עליך להתרכז בלימודים")
